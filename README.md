## Clipboard

[![NuGet](https://img.shields.io/nuget/v/wk.Clipboard.svg)](https://www.nuget.org/packages/wk.Clipboard)

Copy console output to Clipboard without new line

- [x] mac OS
- [ ] Linux
- [ ] Windows

## Installation

```
dotnet tool install -g wk.Clipboard
```

## Usage

```
echo "Hello, world!" | wk-cb
```
